# BrewVibes Cafe & Roastery

BrewVibes is a modern, fully responsive specialty coffee shop website built with HTML5, Bootstrap 5, and vanilla JavaScript.

## Tech Stack
- HTML5
- Bootstrap 5 (CDN)
- Custom CSS (`css/style.css`)
- Vanilla JavaScript (`js/main.js`)

## Features
- SEO-optimized meta tags across all pages
- Custom scrollbar and text selection styling
- Mobile-responsive typography and smooth scrolling
- Responsive navbar with dark/light mode toggle (persisted via localStorage)
- Hero banner on homepage with full-bleed background image
- Testimonials carousel with prev/next navigation
- Filterable menu by category (Coffee, Pastries, Mains)
- Reservation booking form with client-side validation
- Contact form with client-side validation
- Gallery page with responsive image grid
- Events page with upcoming workshops and live nights
- Team page with employee bios and roles
- Scroll-to-top floating button
- IntersectionObserver card fade-in animations
- Nav link underline slide animation on hover
- Accessible aria-labels on interactive elements

## Pages
- `index.html` — Home (hero + testimonials carousel)
- `menu.html` — Menu (filterable cards)
- `reservations.html` — Reservations (booking form + validation)
- `about.html` — About Us (cafe story + image)
- `gallery.html` — Gallery (6-image responsive grid)
- `events.html` — Events (upcoming workshops and live nights)
- `team.html` — Team (employee bios)
- `contact.html` — Contact (location info + message form)

## Getting Started
Open `index.html` in any modern browser — no build tools required.
